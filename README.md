# 通过代码格式

### 目录命名
* 尽可能使用一个单词
* 多个单词间使用减号分隔
* 字母全部小写



### 文件命名
* 每个单词的首字母大写
* 前缀一律大写



### 变量
* 全部小写字母
* 单词之间使用下划线分隔
* 静态变量使用前缀 "s_"
* 成员变量使用前缀 "m_"
* 布尔型使用前缀 "is" 



### 宏
全部大写字母，单词之间使用下划线分隔



### 函数
* 第一个单词首字母小写，其余单词首字母大写


例如：

```
sayHelloToAll()
```


### 空行
* 大括号独占一行
* 函数之间使用2个空行间距
* 函数内，相对明确的逻辑片段留出空行
